About this website

Thanks for browsing my portfolio! Hello and welcome to my website!! This website is a personal project that I'm undertaking to both start building up a portfolio and to learn some web development skills. I'm doing so with the help of The Odin Project, an open-source resource that teaches beginners some rudimentary skills, with the intention of giving some exposure to full-stack web development.


In regards to languages, the Odin Project covers the basics of HTML, CSS, and JavaScript, and then follows into either RoR or .node frameworks (I haven't quite decided which of these I'll adopt just yet, but I'll think about it when I get there!!).


As it stands, I'm still setting up the main display features on the client-side, so there isn't much up and running now except for the 'Film Photography' page - but I'm working on it, and and will have the others up very soon.
